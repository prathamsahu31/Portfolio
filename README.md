A modern, animated, and fully responsive personal portfolio website built to showcase my projects, skills, and educational background.

[View Live Portfolio](https://pratham.app) 

## ✨ Features

- **Modern Dark Theme**: Sleek, minimalist design with glowing accents (Emerald, Indigo, Cyan, Rose).
- **Smooth Animations**: Powered by `framer-motion` for scroll reveals, hover effects, and page transitions.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop viewing.
- **Glassmorphism UI**: Beautiful frosted glass effects on the navigation bar and cards.
- **Interactive Sections**:
  - **Hero**: Dynamic introduction with animated text and a spinning portrait container.
  - **About**: Detailed bio and categorized skill cards.
  - **Education**: Vertical timeline showcasing academic background.
  - **Projects**: Grid layout with hover-to-reveal links for GitHub and live demos.
  - **Contact**: Sleek contact form and direct contact information.

## 🛠️ Tech Stack

- **Framework**: [React 19](https://react.dev/)
- **Build Tool**: [Vite](https://vitejs.dev/)
- **Styling**: [Tailwind CSS v4](https://tailwindcss.com/)
- **Animations**: [Motion (Framer Motion)](https://motion.dev/)
- **Icons**: [Lucide React](https://lucide.dev/)
- **Typography**: Inter & Space Grotesk (Google Fonts)

## 💻 Getting Started

### Prerequisites

Make sure you have Node.js installed on your machine.

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/yourusername/portfolio.git
   ```

2. Navigate to the project directory:
   ```bash
   cd portfolio
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open your browser and visit `http://localhost:3000` (or the port specified in your terminal).
